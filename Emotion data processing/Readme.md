## Repository overview

*readAndSave.m Process the raw data and segment them into waveform and voltage.

*Pic\_loadDataset.m Transform the raw data into corresponding images.



*Train\_image: Read image and train the image-based network

*Test\_image: Test image-based network and show the results

*Train\_sequence: Read sequence data and train the sequence-based network

*Test\_sequence: Test sequence-based network and show the results



*trainedNetwork: Include the trained networks.

*Data parameter: The data used for statistic analysis.
